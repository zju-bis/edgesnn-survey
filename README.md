# Edge Intelligence with Spiking Neural Networks

This repository collects papers on **Spiking Neural Networks (SNNs)** published in influential conferences from 2021 to 2025.  
It serves as the official repository for our survey titled:

> **Edge Intelligence with Spiking Neural Networks**  

The survey systematically reviews SNN works published in top conferences, including **AAAI, ICLR, CVPR, ICCV, IJCAI, MM, ICML, NeurIPS, ECCV**.
